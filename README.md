# LPS-game
Lapis Papyrus Scalpellus Game

This is a javascript game for the classic "Rock Scissor Paper game" with upgrade UI features for the web app.

Translation for interest: <br>
Lapis - Rock/Stone <br>
Papyrus - Paper <br>
Scalpellus - Scalpel/Scissors <br>
